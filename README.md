# Spriha Sharma — Portfolio

A modern, responsive personal portfolio website showcasing my projects, technical skills, and experience as a Computer Science student.

## About This Project

This portfolio is designed to present my work, technical capabilities, and professional background through a clean and interactive interface.

* Fully responsive layout across devices
